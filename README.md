# Language App Backend

## Описание проекта

Backend-часть мобильного приложения **Language App**, разработанная для поддержки образовательной платформы изучения иностранных языков.

Серверная часть отвечает за обработку пользовательских данных, управление контентом, хранение словарей, прогресса обучения и будущих функций авторизации и аналитики.

Проект построен на Node.js с использованием TypeScript и ориентирован на масштабируемую архитектуру API.

---

## Назначение проекта

Основная цель backend-сервиса — обеспечить стабильную и расширяемую серверную инфраструктуру для мобильного приложения.

В рамках проекта реализуется:

* API для мобильного приложения
* управление данными пользователей
* хранение учебных материалов (слова, уроки, упражнения)
* работа с прогрессом обучения
* подготовка к системе авторизации
* seed-скрипты для наполнения базы данных

Проект рассчитан на дальнейшее расширение в сторону полноценной EdTech-платформы.

## Скрипты проекта

```json
"scripts": {
  "dev": "ts-node-dev --respawn src/index.ts",
  "seed": "ts-node src/seedLearn.ts"
}
```

---

## Описание скриптов

### Запуск в режиме разработки

```bash
npm run dev
```

Запускает сервер в режиме разработки с автоматическим перезапуском при изменениях файлов.

Используется `ts-node-dev` для быстрого development workflow.

---

### Заполнение базы данных (seed)

```bash
npm run seed
```

Запускает скрипт начального наполнения базы данных учебными данными:

* слова
* уроки
* тестовые сущности
* начальная структура контента
